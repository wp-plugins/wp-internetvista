Show directly in your wordpress dashboard the performance of your website (uptime and response time).

== Description ==

This plugin allows you to display directly and simply the performance of your website in your wordpress dashboard.

= The website monitoring will help you to: =
* Measure the performance of your website
* Put your web infrastructure under high surveillance
* Prevent breakdowns of your website or your blog
* Prove the quality of your web solutions

= The major advantages of our web monitoring services are: =
* Monitoring provided 24x7x365
* Nothing to install, nothing to develop
* Hosters-independent
* Monitoring managed from your mobile phone
* Monitoring of complex web scenarios
* Monitoring frequency of 1 minute
* Checking the downloaded content
* Alerts by email, sms, twitter, iphone notification,...

== Installation ==

= Add the plugin =
1. Upload the plugin using the Plugin Manager built into WordPress
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your internetVista username/password* 

Enjoy and know your website performance!

*you can register for free on [InternetVista](http://www.internetVista.com/register.htm "InternetVista.com").

= Remove the plugin =
1. Deactivate the plugin through the 'Plugins' menu in WordPress
2. Delete the plugin.
