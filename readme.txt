Uptime is money - This plugin allows you to view directly from your WordPress dashboard the performance of your website (uptime and response time).

== Description ==

This plugin allows you to view directly from your WordPress dashboard the performance of your website (uptime and response time).

= Explanations =

In order to use this plugin you need an (free) account on internetVista.

== Installation ==

1. Upload the plugin using the Plugin Manager built into WordPress
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Freely register on [InternetVista](http://www.internetVista.com/fr/register.htm "InternetVista.com").

= Removal =

1. Deactivate the plugin through the 'Plugins' menu in WordPress
1. Delete the plugin.

== Frequently Asked Questions ==

= Do I need to register on the plateform prior using this plugin? =

Yes, the login/password of your InternetVista's account wil be required in order to retrieve data.
